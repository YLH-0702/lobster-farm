# 龙虾养殖网 - OpenClaw社区

一个专注于OpenClaw AI助手学习、交流和技能分享的社区网站。

## 项目特色
- 🦞 **龙虾养殖隐喻**：将AI助手培养比作龙虾养殖
- 🎓 **学习中心**：OpenClaw教程和最佳实践
- 🤝 **社区交流**：技术讨论和问题求助
- 🛠️ **工具集合**：在线开发工具和资源
- 📦 **技能市场**：用户分享的OpenClaw技能

## 技术栈
- **前端**：Next.js 14 + React + TypeScript
- **样式**：Tailwind CSS + Shadcn UI
- **后端**：Next.js API Routes + Supabase
- **数据库**：PostgreSQL (Supabase)
- **部署**：Vercel

## 快速开始
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

## 环境变量
复制 `.env.example` 到 `.env.local` 并配置：
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
